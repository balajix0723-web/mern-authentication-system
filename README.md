# 🔐 MERN Authentication System

A modern and secure full-stack authentication system built using the MERN stack. This project provides user registration, login, JWT-based authentication, protected routes, and password encryption following industry best practices.

---

## 🚀 Features

- User Registration
- Secure User Login
- JWT Authentication
- Password Hashing with bcrypt
- Protected Routes
- MongoDB Database Integration
- RESTful API
- Responsive React Frontend
- Error Handling & Input Validation

---

## 🛠️ Tech Stack

### Frontend
- React.js

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/mern-authentication-system.git
```

### Navigate to the project

```bash
cd mern-authentication-system
```

### Install dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

---

## ▶️ Run the Project

Backend

```bash
npm run dev
```

Frontend

```bash
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
